Open Connect is a large network of servers across the globe designed to store and transmit heavy video data efficiently to user screens. To keep streams from lagging, Netflix developed Kernel TLS (KTLS) which is a built-in optimization on top of FreeBSD that shifts heavy data encryption directly into the core of their operating system instead of passing it to slower external apps. This allows Netflix to eliminate performance bottlenecks, enabling their servers to transmit 400Gbits/s of secure video traffic to users from all over the world.

            NetApp is an intelligence data infrastructure company that provides unified data storage, integrated data services, and cloud operations solutions to enterprise customers. They utilize FreeBSD as the foundational engine for NetApp ONTAP, a data management software. NetAPP specifically chose FreeBSD for its permissive licensing model which allows NetApp to use its own proprietary technology with FreeBSD’s open-source code without risking software licensing issues. NetApp has also upstreamed over 300 changes and has contributed software like bhyve, FreeBSD’s hypervisor allowing for significant contributions to FreeBSD making a strong relationship with the community.

            The pfSense Project is a free open-source customized distribution of FreeBSD that is specifically tailored for use as a firewall and router that can be managed entirely via a web interface. pfSense uses FreeBSD for handling heavy internet traffic. In utilizing the multiprocessor capabilities for PF in FreeBSD, developers are able to handle the high packets per second rate required by large scale networks. Today, pfSense is widely used in network security, from small home networks, cloud networks, and even large-scale enterprises. Allowing for accessible network security for all users, even those without any prior UNIX knowledge.
        </p>

            RG Nets leverages FreeBSD in constructing rXg, its powerful, multi-functional gateway appliances. They specialize in using FreeBSD for Software Defined Networking (SDN), a technology that allows efficient network management and operation by separating the control plane from the data plane. In taking advantage of FreeBSD’s modular architecture, RG Nets integrates FreeBSD Jails and bhyve in virtualization to handle heavy data processing. Allowing them to seamlessly route massive traffic surges. Similar to NetApp, RG Nets also made significant contributions to pf, FreeBSD’s packet filter firewall, and altq, FreeBSD’s network packet scheduler.
